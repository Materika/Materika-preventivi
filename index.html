📐 Formula di calcolo
Prezzo = CV ÷ (1 − CF% − Margine%)
CV = materiale + stampa + lavorazioni + macchina + setup + trasporto
CF% = costi fissi aziendali (affitto, utenze, personale indiretto)
Es. CV=€50 · CF=35% · Margine=20% → coeff=0,45 → Prezzo=€111,11
